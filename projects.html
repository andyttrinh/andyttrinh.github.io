Python Sudoku Solver

Sudoku Solver created in Python. A backtracking algorithm was used that involved recursion. The algorithm builds up possible candidates to the solution. If it determines that no candidates is possible it abandons the it then moves back to the previous stage. This would continue until it reaches the end of the puzzle.

Web Based Maze Game

A web maze game that uses MatterJS. The game generates a maze randomly every new instance, creating a personal experience for every player. The algorithm uses recursion and randomness to generate the walls, and makes each maze possible.

Movie Comparison Web App

This web App uses the OMDb API as the source for fetching data. It allows the user to select two movies and compare the statistics of both to each other. The better statistics appears in the green color, while the other appears in the yellow color.
